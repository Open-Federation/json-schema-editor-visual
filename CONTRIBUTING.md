# CONTRIBUTING

## Prerequisites

- Node.js: v14.x.x
- Package manager: npm

## Install

```bash
npm install
```

## Develop

Start dev server.

```bash
npm run start
```

Then open `http://127.0.0.1:8082/index.html` in your browser.

Resources will be rebuild when files changed. Refresh page to see updates, HMR not supported.

## Build

```bash
# Build Package
npm run build

# Build Demo Page
npm run demo
```
